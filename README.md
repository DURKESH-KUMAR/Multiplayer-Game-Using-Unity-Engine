# Multiplayer Shooter Game (Unity + Photon)

## Overview
This is a fast-paced **multiplayer shooter game** developed using the **Unity Engine** with **Photon Networking (PUN 2)**. Players can engage in real-time combat with smooth, responsive controls and seamless online multiplayer replication. The game provides both competitive and cooperative modes, ensuring an exciting experience for all players.

## Features
- Real-time multiplayer shooter action using Photon PUN 2.
- Smooth replication of player movements, shooting, and in-game events.
- Competitive and cooperative gameplay modes.
- Optimized networking for minimal lag and high responsiveness.
- Cross-region support for stable online gameplay.

## Technologies Used
- Unity Engine
- Photon PUN 2 (Photon Unity Networking)
- C#
